FAQ

Preguntas frecuentes

Lo que la gente suele preguntar antes de instalar.

¿De verdad es gratis? ¿Cuál es el truco?
Sí, totalmente gratis. No hay versión paga, no hay anuncios, no hay tracking. Es un proyecto open-source con fines educativos mantenido por FofoStudio. Si te gusta, dale una star en GitHub.
¿Necesito permisos de administrador para instalar?
En Windows no: el instalador es per-user, va a tu carpeta de usuario en %LOCALAPPDATA%\Programs\LocalTv. En macOS solo arrastras la app a /Applications.
macOS dice "no se puede abrir, desarrollador no identificado"
Es Gatekeeper, normal en apps no firmadas con Apple Developer ID (firmar cuesta US$99/año, no lo pagamos en este proyecto open-source). Solución una sola vez: clic derecho sobre LocalTv.app → Abrir → Abrir. O por terminal: xattr -dr com.apple.quarantine /Applications/LocalTv.app.
¿Funciona en Mac con M1, M2 o M3?
Sí. El .dmg es universal — corre nativo en Apple Silicon e Intel. macOS 11 (Big Sur) o superior.
¿Y en Linux?
No hay instalador empaquetado todavía, pero el código corre perfecto en modo desarrollo. Clona el repo y corre ./setup.sh. Cualquier distro con Python 3.11+ y Node 18+ funciona.
¿Se actualiza solo?
Por ahora no. Cuando salga una versión nueva, descargas el instalador y lo corres encima. Tus datos (canales, favoritos, configuración) se preservan: viven en una carpeta separada que no se toca al actualizar.
¿Puedo verlo desde mi TV o celular?
Sí, mientras estén en la misma red Wi-Fi. Abre LocalTv en la PC, mira la URL en la ventana de control y reemplaza localhost por la IP local de tu PC (algo como http://192.168.1.20:8765) en el otro dispositivo.
¿Esto es legal?
El software es 100% legal — es un reproductor + agregador. La responsabilidad sobre el contenido que decidas reproducir recae en ti. El proyecto se distribuye con fines educativos. Asegúrate de respetar términos de servicio y derechos de autor de cada contenido.
¿Cómo desinstalo?
Windows: Configuración → Aplicaciones → "LocalTv" → Desinstalar. macOS: arrastra LocalTv.app a la papelera. Para borrar también tus datos: en Windows borra %LOCALAPPDATA%\LocalTv, en macOS borra ~/Library/Application Support/LocalTv.
