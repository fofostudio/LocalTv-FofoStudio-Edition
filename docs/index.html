Preguntas frecuentes

Lo que la gente suele preguntar antes de instalar.

¿De verdad es gratis? ¿Cuál es el truco?

Sí, totalmente gratis. No hay versión de pago, no hay anuncios, no hay tracking. Es un proyecto open-source con fines educativos mantenido por FofoStudio. Si te gusta, dale una estrella en GitHub.

¿Necesito permisos de administrador para instalar?

En Windows no: el instalador es per-user, se instala bajo tu carpeta de usuario en %LOCALAPPDATA%\Programs\LocalTv. En macOS solo arrastras la app a /Applications (te puede pedir tu contraseña la primera vez, dependiendo de tu cuenta).

macOS dice "no se puede abrir, desarrollador no identificado". ¿Qué hago?

Eso es Gatekeeper, normal en apps no firmadas con Apple Developer ID (firmar cuesta US$99/año, no lo hacemos en este proyecto open-source). Solución, una sola vez: clic derecho sobre LocalTv.app → Abrir → Abrir en el diálogo. Listo, queda autorizada para siempre. Como alternativa por terminal: xattr -dr com.apple.quarantine /Applications/LocalTv.app.

¿Funciona en mi Mac con M1 / M2 / M3?

Sí. El .dmg incluye una build universal que corre nativa en Apple Silicon e Intel. Se instala desde macOS 11 (Big Sur) en adelante.

¿Y en Linux?

No hay un instalador empaquetado, pero el código corre perfecto en Linux en modo desarrollo. Clona el repo y corre ./setup.sh. La app es 100% web — Vite + FastAPI — así que cualquier distro con Python 3.11+ y Node 18+ la puede correr.

¿Se actualiza solo?

Por ahora no. Cuando saquemos una versión nueva, descargas el instalador desde aquí y lo corres encima. Tus datos (canales, favoritos, configuración) se preservan: viven en una carpeta separada del usuario que no se toca al actualizar.

¿Puedo verlo desde mi TV o celular?

Sí, mientras estén en la misma red Wi-Fi. Abre LocalTv en tu PC, mira la URL en la ventana de control y reemplaza localhost por la IP local de tu PC (algo como http://192.168.1.20:8765) en el otro dispositivo.

¿Esto es legal?

El software es 100% legal — es un reproductor + un agregador. La responsabilidad sobre el contenido que decidas reproducir recae en ti. El proyecto se distribuye con fines educativos. Asegúrate de respetar los términos de servicio de las plataformas que uses y los derechos de autor de cada contenido.

¿Cómo desinstalo?

Windows: Configuración → Aplicaciones → busca "LocalTv" → Desinstalar. macOS: arrastra LocalTv.app a la papelera. Si quieres borrar también tus datos: en Windows borra %LOCALAPPDATA%\LocalTv, en macOS borra ~/Library/Application Support/LocalTv.
